# 🛡️ Helper Bot (Report) 1.0

### 🚀 Официальный сканер безопасности экосистемы helpings bots.

Этот бот предназначен для очистки Telegram от вредоносного контента. 
Он использует алгоритмы анализа для выявления угроз психике, мошенничества и токсичности.

---

## ⚡ Функции:
* 🔍 **Deep Scan:** Глубокий технический осмотр ссылок (30-60 секунд).
* 📊 **Verdict System:** Выставление баллов опасности (Toxic, Scam, 18+).
* 🚨 **Report System:** Мгновенная отправка заявок на снос администраторам.
* 🛡️ **Anti-Toxic Shield:** Защита пользователей от опасного контента.

## 🛠 Технологии:
* **Language:** Python 3.x
* **Library:** aiogram 3.x
* **Engine:** Daster Engine v1.0
